隐私政策

最近更新：2026 年 4 月 18 日

1. 引言

AnPin AI（以下简称"我们"）重视您的隐私保护。本隐私政策说明了我们如何收集、使用、存储和保护您的个人信息。使用我们的服务即表示您同意本政策的内容。

2. 信息收集

我们收集以下类型的信息：

  • 账户信息：注册时提供的邮箱地址和密码（加密存储）。
  • 使用数据：API 调用记录、Token 用量统计、请求时间等。
  • 支付信息：充值记录和订单信息（不存储银行卡号等敏感支付信息）。
  • 技术信息：IP 地址、浏览器类型、访问时间等技术日志。

3. 信息使用

我们收集的信息用于以下目的：

  • 提供、维护和改善我们的服务
  • 处理充值和账户管理
  • 生成使用统计和账单
  • 保障系统安全和防止滥用
  • 向您发送服务相关的通知

4. API 请求数据

关于您通过 API 发送和接收的内容（包括提示词和模型输出）：

  • 我们作为 API 网关仅进行请求转发，不会长期存储您的请求内容。
  • 为排查故障和统计用量，系统可能在短时间内保留请求元数据（不含内容）。
  • 您的请求内容将被转发至对应的上游 AI 模型提供商，其数据处理方式受各提供商自身隐私政策约束。

5. 信息共享

我们不会向第三方出售您的个人信息。在以下情况下，我们可能会共享您的信息：

  • 为提供服务而将 API 请求转发至上游模型提供商
  • 处理支付交易时与支付服务商共享必要的订单信息
  • 应法律法规要求或执法机关的合法请求
  • 在您明确同意的情况下

6. 数据安全

我们采取合理的安全措施保护您的信息：

  • 所有数据传输采用 TLS/HTTPS 加密
  • 密码使用单向散列算法加密存储
  • API Key 使用安全的生成和存储机制
  • 定期进行安全审计和漏洞排查
  • 实施访问控制和权限管理

7. 数据保留

我们将在提供服务所必需的期限内保留您的信息。账户注销后，我们将在合理期限内删除或匿名化您的个人信息，但法律法规要求保留的除外。

8. 您的权利

您拥有以下权利：

  • 访问和查看您的个人信息
  • 更正不准确的个人信息
  • 请求删除您的账户和数据
  • 导出您的使用数据
  • 撤回对数据处理的同意

9. Cookie 使用

我们使用 Cookie 和类似技术来维持您的登录状态和改善使用体验。您可以通过浏览器设置管理 Cookie 偏好，但禁用 Cookie 可能影响部分功能的使用。

10. 未成年人保护

本服务不面向 18 周岁以下的未成年人。我们不会故意收集未成年人的个人信息。如发现误收集了未成年人的信息，我们将及时删除。

11. 政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面公布，重大变更将通过邮件或站内通知告知您。建议您定期查阅本政策。

12. 联系我们

如您对本隐私政策有任何疑问或需要行使您的数据权利，请通过控制台内的联系方式与我们取得联系。
