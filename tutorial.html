快速上手

只需三步，即可开始使用 AnPin AI 的大模型 API 服务

1
注册账号

点击右上角「控制台」按钮，使用邮箱注册一个 AnPin AI 账号。注册完成后登录控制台。

💡 注册后即赠送免费额度，可直接开始体验所有模型。
2
创建 API Key

登录控制台后，进入「API Keys」页面，点击「创建 API Key」按钮。创建成功后，请妥善保存你的 Key，它仅在创建时显示一次。

你的 API Key 格式类似：

sk-anpin-xxxxxxxxxxxxxxxxxxxx
3
开始调用

AnPin AI 完全兼容 Anthropic / OpenAI 官方 API 格式，只需将 base_url 替换为 AnPin AI 的地址即可。

Python (Anthropic SDK) 示例：

import anthropic

client = anthropic.Anthropic(
    api_key="sk-anpin-xxxx",
    base_url="https://your-domain.com"
)

message = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "你好，请介绍一下自己"}
    ]
)
print(message.content[0].text)

cURL 示例：

curl https://your-domain.com/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: sk-anpin-xxxx" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "claude-sonnet-4-20250514",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'
4
充值与查看用量

在控制台「充值」页面选择套餐进行充值。在「个人中心」可以查看实时余额和详细用量统计。

所有模型按实际 Token 用量计费，价格详见 模型广场。

5
在 Claude Code 中使用

如果你使用 Claude Code（CLI 工具），可以这样配置：

# 设置环境变量
export ANTHROPIC_API_KEY="sk-anpin-xxxx"
export ANTHROPIC_BASE_URL="https://your-domain.com"

# 启动 Claude Code
claude
💡 也可以在 Claude Code 的设置中配置 API Base URL，无需每次设置环境变量。
