API 文档

AnPin AI 完全兼容 Anthropic Messages API 格式，可无缝切换使用。

简介

AnPin AI 提供统一的 API 网关服务，通过一个 API Key 即可访问 Claude、GPT、Gemini 等多种大模型。API 接口与 Anthropic 官方完全兼容，现有代码仅需修改 Base URL 即可接入。

认证方式

所有 API 请求都需要在 HTTP Header 中携带 API Key 进行认证：

x-api-key: sk-anpin-xxxxxxxxxxxx

也支持 Bearer Token 格式：

Authorization: Bearer sk-anpin-xxxxxxxxxxxx

Base URL

将请求的 Base URL 替换为你的 AnPin AI 部署地址：

https://your-domain.com
⚠️ 请勿在客户端或前端代码中暴露 API Key。API Key 应仅在服务端使用。

Messages API

发送消息并获取 AI 回复，兼容 Anthropic Messages API。

请求

项目值
方法POST
路径/v1/messages
Content-Typeapplication/json

请求体参数

参数类型必填说明
modelstring是模型名称，如 claude-sonnet-4-20250514
messagesarray是对话消息列表
max_tokensinteger是最大输出 Token 数
systemstring否系统提示词
temperaturenumber否采样温度 (0-1)
streamboolean否是否启用流式输出

示例请求

curl https://your-domain.com/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: sk-anpin-xxxx" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "claude-sonnet-4-20250514",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "什么是 API 网关？"}
    ]
  }'

示例响应

{
  "id": "msg_xxxx",
  "type": "message",
  "role": "assistant",
  "content": [
    {
      "type": "text",
      "text": "API 网关是..."
    }
  ],
  "model": "claude-sonnet-4-20250514",
  "usage": {
    "input_tokens": 12,
    "output_tokens": 150
  }
}

查询可用模型列表。

项目值
方法GET

Streaming（流式输出）

在请求中设置 "stream": true 即可启用 Server-Sent Events (SSE) 流式输出，实时获取模型生成的内容。

curl https://your-domain.com/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: sk-anpin-xxxx" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "claude-sonnet-4-20250514",
    "max_tokens": 1024,
    "stream": true,
    "messages": [
      {"role": "user", "content": "写一首短诗"}
    ]
  }'

错误处理

API 返回标准 HTTP 状态码，错误时附带 JSON 错误信息：

状态码说明
400请求参数错误
401API Key 无效或未提供
403权限不足或账户已被禁用
429请求频率超限，请稍后重试
500服务器内部错误
529上游过载，请稍后重试

速率限制

速率限制取决于你的账户等级和套餐配置。超出限制时会返回 429 状态码。建议在代码中实现指数退避重试机制。

SDK 支持

由于完全兼容 Anthropic API，你可以直接使用官方 SDK：

语言SDK安装命令
Pythonanthropicpip install anthropic
Node.js@anthropic-ai/sdknpm install @anthropic-ai/sdk
Goanthropic-gogo get github.com/anthropics/anthropic-sdk-go

只需在初始化 SDK 时指定自定义 base_url 即可。
